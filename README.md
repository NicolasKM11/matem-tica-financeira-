
# Matemática Financeira Aplicada — V8

Versão ampla em Streamlit com:
- porcentagem e taxa real
- taxas proporcionais
- taxas equivalentes
- taxa nominal x efetiva
- juros simples
- juros compostos
- desconto racional simples
- desconto comercial simples
- desconto racional composto
- desconto comercial composto
- custo efetivo no desconto
- séries uniformes: postecipada, antecipada, diferida e perpetuidade
- entrada + parcelas / taxa implícita
- renegociação
- fluxo de caixa irregular: VPL/NPV e TIR/IRR
- comparador de alternativas

## rodar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```
