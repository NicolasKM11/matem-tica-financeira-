
# Matemática Financeira Aplicada — versão reduzida

App em Streamlit focado apenas nestes tipos de exercícios:
- taxa nominal x taxa efetiva
- taxa real
- desconto comercial simples
- custo efetivo do desconto com saldo médio e despesa
- séries uniformes: postecipada, antecipada, diferida e perpetuidade
- fluxo de caixa irregular com VPL/NPV e TIR/IRR
- comparação entre bancos

## rodar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```
