# App de Matemática Financeira

App em Streamlit pronto para hospedar.

## Rodar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Hospedar online
### Streamlit Community Cloud
1. Suba os arquivos `app.py` e `requirements.txt` para um repositório no GitHub
2. Entre no Streamlit Community Cloud
3. Clique em **New app**
4. Selecione o repositório e o arquivo `app.py`
5. Deploy

### Render
Use também o arquivo `render.yaml`.
